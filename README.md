# The Gates

> AI characters guard doors. Students write to persuade them open. The characters won't budge unless you've done the reading.

## What This Is

An educational game built on persuasive writing and primary historical sources. Students encounter AI characters throughout a physical library (State Library Victoria). Each character holds a position grounded in a real primary source. The student must find the source, read it, and write an argument that genuinely engages with the material. Generic rhetoric bounces off. You have to have read the thing.

The game is being developed for [Tech World](https://github.com/enspyrco/tech_world), a multiplayer 2D game platform in Flutter with AI characters, real-time chat, and an in-game challenge system.

## How It Works

1. A student encounters a guard — an AI character with a name, a historical moment, and a position
2. The guard's beliefs are grounded in a specific primary source (Galileo's letter, Douglass's speech, a mill girl's diary)
3. The student writes an argument to persuade the guard to open the gate
4. The guard responds in character — pushes back, asks questions, challenges weak reasoning
5. The gate opens only when the argument demonstrates genuine engagement with the source material

The guards are not strawmen. Their positions are the strongest version of what a real person in that role would believe. If the student feels they've beaten a weak opponent, the game has failed.

## The 10 Levels

The game teaches persuasive writing through a progression of increasingly sophisticated challenges:

| Level | Guard | Skill |
|-------|-------|-------|
| 1 | The Sympathetic Gate | Clarity of purpose |
| 2 | The Sceptic | Evidence |
| 3 | The Believer | Empathy before argument |
| 4 | The Bureaucrat | Institutional persuasion |
| 5 | The Opponent | Engaging the strongest counter-argument |
| 6 | The Mirror | Critiquing your own side's weak arguments |
| 7 | The Historian | Historical empathy + primary sources |
| 8 | The Coalition | Persuading multiple stakeholders |
| 9 | The Locked Door | Recognising when persuasion won't work |
| 10 | The Final Gate | Writing outside your comfort zone |

See [GATES.md](GATES.md) for full descriptions of each level.

## The Sources

63 primary historical sources across 8 themes, curated for 14-18 year olds:

- **I. Power and Its Justifications** — Code of Hammurabi to Leopold II
- **II. Resistance and Revolution** — Spartacus to Walesa
- **III. Science and the Overthrow of Certainty** — Galileo to Szilard
- **IV. War and Its Witnesses** — Thucydides to Tim O'Brien
- **V. Daily Life and Hidden Voices** — Pliny to the Partition of India
- **VI. Ideas That Changed the Frame** — Plato to Havel
- **VII. The View from Here** — Turing to Mahsa Amini
- **VIII. The Talmudic Pairs** — structured confrontations between sources

Each source has difficulty ratings, pairing suggestions, and notes for mentors ("Mirandas"). See [SOURCES.md](SOURCES.md) for the full list.

## The Guard Prompts

The `guards/` directory contains ready-to-use AI system prompts for Level 7 (The Historian) — one for each of the 63 primary sources. Each prompt:

- Establishes a character grounded in a specific historical moment
- Gives them a position based on the primary source
- Instructs them to push back against weak arguments
- Makes them genuinely moveable by arguments that engage with the source material
- Prevents anachronistic reasoning from working ("human rights!" doesn't persuade a Babylonian judge)

The guard prompts were generated by Claude Opus 4.6 based on the source descriptions and game design principles.

## The Library Connection

The game is designed for physical libraries. The digital map mirrors the library floor plan. AI characters are placed near the sections where their source material lives. The game sends the student to the catalogue. The catalogue sends them to a shelf. They read because they need to win.

Librarians play the Miranda role (*The Diamond Age*, Neal Stephenson) — they select sources, place gates, observe student progress, and use their knowledge of the collection as a superpower the AI doesn't have.
